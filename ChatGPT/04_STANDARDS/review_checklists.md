# Review Checklists

## Code Review

- [ ] **Functionality**: Does it do what it's supposed to do?
- [ ] **Standards**: Does it follow `04_STANDARDS/coding.md`?
- [ ] **Tests**: Are there unit tests? Do they pass?
- [ ] **Simplicity**: Can this be simpler?
- [ ] **Security**: Are there vulnerabilities?

## Project Review

- [ ] **Alignment**: Does this serve a Role or Project objective?
- [ ] **Leverage**: Is there a reusable asset created?
- [ ] **Documentation**: Is the `00_README` or `context.md` updated?
- [ ] **Lessons**: Is the `06_LESSONS` updated if this was a fix?

## Document Review

- [ ] **Clarity**: Is it unambiguous?
- [ ] **Typos**: Is it professional?
- [ ] **Format**: Does it render correctly in Markdown?
